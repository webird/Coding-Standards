#CSS编码规范

* 样式采用：reset.css + comons.css + ${page}.css 形式。reset.css点击[这里](http://developer.yahoo.com/yui/reset/)下载；
* 样式头部需要写明作者和创建日期；

```css
/*
 * @author tiger
 * @date 2013-07-15
 */
body {
    ...
}
```

* 样式文件统一使用四个空格缩进，禁止使用TAB缩进；

```css
/* 不推荐 */
body {
  color:red;
}
/* 推荐 */
body {
    color:red;
}
```

* 统一使用小写命名，单词间使用<code>-</code>分割，同时不要使用无意义的命名，禁止使用拼音命名；

```css
/* 不推荐 */
.subNav {...}
.zidaohang {...}
/* 推荐 */
sub-nav {...}
```
